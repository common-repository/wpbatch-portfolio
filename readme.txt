== Description ==

Very Simple and Clean, Responsive Portfolio Plugin
The columns will be set up automatically with your screen as It's developed on masonry and grid style.

Demo: <a href="http://dreamwebit.com/wpbatch-portfolio">WPBatch Portfolio Plugin</a>

Please check the demo

== Installation ==



1. Upload `wpbatch-portfolio.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see a new menu in the dashboard "WPBatch Portfolio"... add some portfolios with title, description and categories from there
4. now just add this shortcode anywhere in the page or post
<pre>[wpbatch_portfolio]</pre>
or Place <pre>&lt;?php do_shortcode('[wpbatch_portfolio]'); ?&gt;</pre> in your templates php files

Good Luck
